═══════════════════════════════════════════════════════════════
📋 الخطوات المطلوبة:
═══════════════════════════════════════════════════════════════

الخطوة 1: افتح Hostinger Panel
──────────────────────────────────────────────────
🌐 افتح المتصفح واذهب إلى:
   https://hpanel.hostinger.com

🔐 سجل الدخول بحسابك


الخطوة 2: افتح File Manager
──────────────────────────────────────────────────
1. من القائمة الجانبية، اختر "Files"
2. ثم اضغط على "File Manager"
3. أو اذهب مباشرة إلى:
   https://hpanel.hostinger.com/websites/ielts.sbs/file-manager


الخطوة 3: انتقل إلى مجلد dist
──────────────────────────────────────────────────
1. اضغط مرتين على مجلد: public_html
2. اضغط مرتين على مجلد: dist
3. يجب أن تكون الآن في:
   /public_html/dist/


الخطوة 4: احذف index.html القديم
──────────────────────────────────────────────────
1. ابحث عن ملف: index.html
2. اضغط بالزر الأيمن عليه
3. اختر: Delete
4. أكد الحذف


الخطوة 5: ارفع index.html الجديد
──────────────────────────────────────────────────
1. اضغط على زر "Upload" في الأعلى
2. اضغط "Select Files"
3. انتقل إلى:
   C:\developer\QIIC\frontend\dist\
4. اختر: index.html
5. اضغط "Open"
6. انتظر حتى يكتمل الرفع (شريط أخضر)
7. اضغط "Close" بعد الانتهاء


الخطوة 6: انتقل إلى مجلد assets
──────────────────────────────────────────────────
1. اضغط مرتين على مجلد: assets
2. يجب أن تكون الآن في:
   /public_html/dist/assets/


الخطوة 7: احذف ملفات JavaScript القديمة
──────────────────────────────────────────────────
ابحث عن هذه الملفات واحذفها (إن وجدت):

❌ index-B5uxdLf-4.js
❌ index-CFVA53ab.js  
❌ index-BrSuvDL4.js

لكل ملف:
1. اضغط بالزر الأيمن
2. اختر Delete
3. أكد الحذف


الخطوة 8: ارفع ملفات JavaScript و CSS الجديدة
──────────────────────────────────────────────────
1. اضغط على زر "Upload"
2. اضغط "Select Files"
3. انتقل إلى:
   C:\developer\QIIC\frontend\dist\assets\
4. اختر هذين الملفين معاً (اضغط Ctrl + Click):
   ✅ index-DNtHMks8.js
   ✅ index-Bsy0WF_g.css
5. اضغط "Open"
6. انتظر حتى يكتمل الرفع (قد يستغرق دقيقة)
7. اضغط "Close"


الخطوة 9: تحقق من الملفات المرفوعة
──────────────────────────────────────────────────
في مجلد assets، يجب أن ترى:

✅ index-DNtHMks8.js (312.81 KB)
✅ index-Bsy0WF_g.css (177.72 KB)
✅ react-core-CyvzqkFf.js
✅ react-router-I5Oko-29.js
✅ socket-io-CUkmNz_4.js
✅ icons-vQu4F1pA.js

يجب ألا ترى:
❌ index-B5uxdLf-4.js
❌ index-CFVA53ab.js
❌ index-BrSuvDL4.js


═══════════════════════════════════════════════════════════════
✅ اختبار الموقع بعد الرفع:
═══════════════════════════════════════════════════════════════

الخطوة 1: افتح متصفح Incognito
──────────────────────────────────────────────────
اضغط: Ctrl + Shift + N (Chrome) أو Ctrl + Shift + P (Firefox)


الخطوة 2: اذهب إلى صفحة Admin Login
──────────────────────────────────────────────────
افتح: https://ielts.sbs/admin/login


الخطوة 3: افتح Developer Tools
──────────────────────────────────────────────────
اضغط: F12


الخطوة 4: تحقق من Network Tab
──────────────────────────────────────────────────
1. اضغط على تبويب "Network"
2. اضغط F5 لتحديث الصفحة
3. ابحث عن ملف: index-DNtHMks8.js
4. يجب أن يظهر في القائمة مع Status: 200


الخطوة 5: تحقق من عدم وجود أخطاء
──────────────────────────────────────────────────
✅ يجب أن ترى صفحة تسجيل الدخول بشكل طبيعي
❌ يجب ألا ترى رسالة "⚠️ الوصول غير مصرح به"


═══════════════════════════════════════════════════════════════
🎯 النتيجة المتوقعة:
═══════════════════════════════════════════════════════════════

إذا نجح كل شيء:
✅ صفحة Admin Login تظهر بشكل طبيعي
✅ حقول اسم المستخدم وكلمة المرور تعمل
✅ لا توجد رسالة "الوصول غير مصرح به"
✅ يمكنك تسجيل الدخول بنجاح


إذا لم يعمل:
1. امسح Cache المتصفح: Ctrl + Shift + Delete
2. اضغط Ctrl + F5 عدة مرات
3. تأكد من رفع index.html بشكل صحيح
4. تأكد من وجود index-DNtHMks8.js في assets
